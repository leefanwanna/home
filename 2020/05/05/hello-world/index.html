我的第一篇博客
发表于 2020-05-05| 更新于 2020-05-21
字数总计:325|阅读时长: 1 分钟
|阅读量:

　　Hello!这是我的第一篇博客。忙活了很久好歹是把它部署成功了。如果要用一句话来记录我的心情，我会说，嘿！还记得你第一次敲出的 Hello world 吗！那是我第一次推开编程的大门。是一切开始的时刻。

　　如果说学计算机最大的快乐莫过于无数次打开新世界的大门，那么最大的痛苦就是在新世界里无所适从。这两天，尝试搭建了七八个博客，GitHub 的仓库建了又删，见识了各种千奇百怪的错误。但出错碰壁总是好的，又学到了不少知识。而且谁让我就是爱折腾呢？这个博客将会用于记录自己的学习历程，以及我的各种想法。

　　我一直很庆幸自己能够在报志愿的时候选择了计算机。这样，我才在最有活力的年纪对无数个新世界道一声 Hello world 。我也同样庆幸自己生在了这样一个开源的时代，无论是 GitHub 这样一个开源的宝库，还是各位巨巨们写的博客，这是我人生中第一次， 真正直观且切实的体会到知识汪洋的壮阔。
